Riktlinjer för iDOC=

  1. Enbart dokumentation relaterad till Commodores åttabitarsdatorer. Vi accepterar dokumentationer från andra plattformar så länge de speglar saker som gjorts för Commodores åttabitarsdatorerdatorer. Om den enda dokumentation som finns tillgänglig är för andra maskiner, så kan de läggas in här, men de kommer att tas bort när originalen för Commodoredatorer blir tillgängliga.
  2. Det primära formatet för iDOC=-dokumentation är råtext, vilket betyder att grafik måste konverteras till ASCII-konst eller beskrivningar som information, eller kanske t.o.m känslor. Om grafik krävs, kan andra publiceringsformat vara nödvändiga, vänligen Mike Naberezny på mike@naberezny.com för mer information.
  3. Främst enbart icke-engelska dokument. Eftersom det ursprungliga Project 64 redan finns för engelska dokumentationer, ser jag ingen anledning att inkludera dem även här. Vissa engelska översättningar kan dock inkluderas, och engelska dokument för maskiner som inte Project 64 täcker.
  4. "Hacker-dokumentationer" tolereras. N:te generationens etexter från okänt ursprung inkluderas inom projektet, men kommer troligen att ersättas med etexter från kända original när de blir tillgängliga.
  5. iDOC= distribuerar inte spel eller annan programvara, enbart dokumentation. Merparten av den programvara som omnämns här finns troligen redan på någons vävsida, på Arnold eller någon annanstans.
  6. iDOC= erkänner dokumentationernas upphovsrätt. Eftersom de flesta Commodore åttabitars-saker inte längre tillverkas tror vi inte att det ligger någon intressekonflikt i denna fria distribution av dokument. Om du, trots allt, skulle ha någonting att protestera mot denna distribution, var vänlig kontakta Mike Naberezny på mike@naberezny.com, så att vi kan få rätsida på det hela. Om du är upphovsrättsinnehavare för någon av de tillgängliga dokumentationerna och inte vill att det ska distribueras här, så kommer vi naturligtvis att ta bort det så snart som möjligt.
  7. iDOC= är ett ideellt projekt.

Senast uppdaterad den 2008-05-25
